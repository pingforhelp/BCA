# Pimgforhelp/BCA
A `.md` notes of BCA (Sem.1) as per AU Textbooks. 
# Introduction
A web-based notes of BCA. Updated in **real-time**. Hosted on Github and Cloudflare pages. Rendered in Vitepress a modern documentation framework. Customize CSS for Callouts, Underline, and, Text Highlighting.

# Quick Start
Clone this repo:
`git clone https://github.com/pingforhelp/BCA/`

Change Path:
`cd BCA`

Install Dependencies
``` bash
npm install
npm run build
```
Run Locally
``` bash
npm run dev

> bca-notes@1.0.0 dev
> vitepress dev docs

  vitepress v1.6.4

  ➜  Local:   http://localhost:5173/ # Local
  ➜  Network: use --host to expose
  ➜  press 'h' to show help
  ```
