# Computer Fundamentals (with Answers)

1. What is a computer?
   A computer is an electronic device that accepts input, processes data, and gives output.

2. What are the basic components of a computer?
   Input unit, CPU, memory, and output unit.

3. What is hardware?
   Physical parts of a computer that we can touch are called hardware.

4. What is software?
   Programs and instructions that tell the computer what to do are called software.

5. Difference between hardware and software?
   Hardware is physical, software is not physical.

6. What is CPU?
   CPU is the brain of the computer that processes data.

7. Parts of CPU?
   ALU, Control Unit, and Registers.

8. What is RAM?
   RAM is temporary memory used to store data while working.

9. Is RAM volatile?
   Yes, RAM is volatile memory.

10. What is ROM?
    ROM is permanent memory that stores startup instructions.

11. Assembling & Disassembling of Computer Systems

12. What is assembling?
    Assembling means connecting all computer hardware parts together.

13. What is disassembling?
    Disassembling means removing computer parts safely.

14. Name internal components of CPU.
    Motherboard, RAM, processor, hard disk, power supply.

15. What is motherboard?
    It is the main circuit board where all components are connected.

16. What is SMPS?
    SMPS supplies power to all computer components.

17. What precautions should be taken while assembling?
    Switch off power, handle parts carefully, avoid static electricity.

18. Computer Networks & Topology

19. What is a computer network?
    A network connects multiple computers to share data.

20. What is network topology?
    The arrangement of computers in a network is called topology.

21. Name different network topologies.
    Star, Bus, Ring, Mesh.

22. Which topology is commonly used in colleges?
    Star topology.

34. MS Excel (Spreadsheet)

35. What is MS Excel?

36. What is a cell?
    Intersection of row and column is called a cell.

37. What is a formula?
    An equation used to calculate values.

38. What is a function?
    Predefined formula in Excel is called a function.

39. IF, AND, OR, IFERROR

40. What is IF function?
    It checks a condition and gives result.

41. Syntax of IF function?
    =IF(condition, true value, false value)

42. What is AND function?
    Returns TRUE if all conditions are true.

43. What is OR function?
    Returns TRUE if any one condition is true.

44. What is IFERROR?
    It handles errors and shows alternate value.

45. Lookup Functions

46. What is VLOOKUP?
    Used to search data vertically in a table.

47. What is HLOOKUP?
    Used to search data horizontally.

48. What is XLOOKUP?
    Advanced lookup function used instead of VLOOKUP.

49. What is INDEX function?
    Returns value from a specified position.

50. What is MATCH function?
    Finds position of a value in a range.
